# Github1sExtension

One second to read GitHub code with VS Code.

Screenshot:

![](./image/image.png)

Supported browser:

- `Chrome`
- `Chromium`
- `Brave`
- `FireFox`(Please switch branch `FireFox`)

# Install

Drag the `crx` file to the browser.
