# Github1s扩展

一键在`VSCode`中打开`Github`仓库，灵感基于[github1s](https://github.com/conwnet/github1s)。

效果：

![](./image/image.png)

支持浏览器：

- `Chrome`
- `Chromium`
- `Brave`
- `FireFox`（请切换到`FireFox`分支）

# 安装

直接拖拽`crx`安装即可。
