# Github1s扩展

一键在`VSCode`中打开`Github`仓库，灵感基于[github1s](https://github.com/conwnet/github1s)。

效果：

![](./image/image.png)

支持浏览器：

- `Chrome`/`Chromium`/`Brave`（请切换到`main`分支）
- `FireFox`

# 安装

- `clone`仓库
- 打开火狐
- 输入`about:debugging#/runtime/this-firefox`
- 点击`Load Temporary Add-on`
- 选择`clone`后的`manifest.json`
